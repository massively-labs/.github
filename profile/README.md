# Massively Labs

## Composable GPU computing in Rust.

Massively Labs is an open-source organization building reusable parallel algorithms and experimental GPU-accelerated systems in Rust.

At the center of the organization is massively, a multi-platform library for parallel and graph algorithms built on CubeCL.

Our goal is to make GPU programming less about writing an isolated kernel for every problem and more about composing reusable algorithms over device-resident data.

## What We Build

Massively Labs has two closely connected roles:

### 1. Develop the core abstractions
Build portable vector, segmented, and graph algorithms for GPU workloads.

### 2. Test them on real problems
Use databases, simulations, geospatial processing, graph computing, and numerical experiments to discover which abstractions are actually reusable.
